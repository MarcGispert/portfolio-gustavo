# Design Token Generation Rules
- Source: Use `design-system-info.md` as the ONLY source of truth for values.
- Architecture: 2-Tier (Primitives and Semantic).
- Categories: Colors, Typography (Montserrat & Inter), Spacing (4/8px grid), Shadows (Nav), and Buttons.
- Format: W3C JSON for Token Studio.
- Special Instructions: Include mobile scales (e.g., H1 Mobile 40px) as separate tokens or sets.
